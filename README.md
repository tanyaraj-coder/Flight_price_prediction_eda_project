# Flight Price EDA

Dataset: Flight Price Dataset

Objective:
To analyze flight ticket prices and understand how factors like airline, stops, travel class, duration, and booking time affect pricing.

Key Analysis Performed:

+ Data cleaning and preprocessing

+ Exploratory analysis on airlines, sourceâ€“destination routes, and ticket prices

+ Analysis of price variation based on stops, travel class, and departure time

+ Study of booking patterns using Days Left feature

+ Visualization of price trends across airlines and routes

Key Insights:

+ Business class tickets are significantly more expensive than Economy class

+ Non-stop flights generally cost more than flights with one or more stops

+ Ticket prices increase as the travel date approaches

+ Certain airlines consistently have higher average prices

+ Longer flight duration often leads to higher ticket prices
